Intergrated Onboarding Guide

What is this?

The Integrated Onboarding Guide is a simulated workspace which allows employees to familiarize themselves with the tools and process of their daily workflow to reduce any confusion when working on the actual tasks.

project overview

The Challenge

Create and preserve an accessible, comprehensive and clear training programme for learning task procedures, in order to reduce time spent picking up tasks.

The Solution

  • Researching the target users by doing ethnographic research
  • Creating a persona and respective user journey from ethnographic research
  • Running Participatory Workshop
  • Generating multiple iterations of the mockup informed by user testing and insight

My Role

UI/UX Designer

The Tasks

  • Conducting ethnographic research
  • Creating personas
  • Generating design focus for technological solution.
  • Facilitating Participatory workshop to extract end-user insights.
  • Creating Interactive Mockup
