project overview

The Challenge

We needed to design an interface as simple as possible while maximizing functionality, considering that users are professionals.

The Solution

  • Creating a mockup of the app from an imagined persona
  • Finding particiants to run user testing
  • Generating multiple iterations of the mockup informed by user testing

My Role

UI/UX Designer

The Tasks

  • Provide avenues for professional musicians to access industry standard tools & services without obstructive barriers of connections or nepotism;
  • Localize and centralize a otherwise inconsistent set of interactive and transactional processes between any given industry professionals.
  • Allow studio owners and professionals to re-establish a name for themselves within an industry hierarchy fractured and divided by a post-pandemic world.
  • Set a precedent for a industry facet largely ignored and overlooked by the application world.
